# BossRaid_API

BossRaid_PVE Game contents API

## Commit Convention

- `<동사원형>`: `<본문내용>` `<작성/수정/삭제>`

```
- Init : 처음 커밋 및 환경 설정
- Feat : 새로운 API에 대한 기능 추가
- Modify : 기존 API에 대한 기능 고도화 및 코드 수정 (버그x)
- Chore : 빌드 업무 수정, 패키지 매니저 수정, 그 외 주석수정 etc.

- Docs : 문서 수정
- Style : 코드 포맷팅, 코드 변경이 없는 경우, linting
- Test : 테스트 코드, 리팩터링 테스트 코드

- Refactor : 코드 리팩터링
- Fix : 버그 수정
```
